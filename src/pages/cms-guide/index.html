博客内容管理系统

实现网站内容的在线编辑和自动发布

迁移至 Netlify 的优势

在网站中直接编辑内容，无需手动管理文件

内容编辑后自动推送到 GitHub 仓库

支持多人协作编辑

提供媒体文件上传功能

实时预览编辑效果

迁移步骤

1

部署到 Netlify

访问 https://app.netlify.com/signup 创建账户

2

连接 GitHub 仓库

在 Netlify 中选择 "New site from Git" 并连接到您的 GitHub 仓库

3

启用身份验证服务

在 Netlify 控制面板中，进入 "Settings" → "Identity"，启用 Identity 服务

4

启用 Git Gateway

在 "Settings" → "Identity" → "External providers" 中启用 Git Gateway

5

访问 CMS 管理界面

访问您的网站并添加 /admin/ 路径，例如: https://your-site.netlify.app/admin/

使用说明

迁移完成后，您将能够：

  • 通过 /admin/ 访问内容管理系统
  • 直接在线创建和编辑帖子、日记和相册
  • 编辑后内容会自动推送到 GitHub 仓库
  • 无需手动管理 Markdown 文件
  • 支持图片上传和管理

提示

完成迁移后，您可以通过 https://your-site.netlify.app/admin/ 访问内容管理系统。 所有内容编辑都将直接同步到您的 GitHub 仓库，无需手动操作。
